# MouseLock
[![Download count](https://img.shields.io/endpoint?url=https://qzysathwfhebdai6xgauhz4q7m0mzmrf.lambda-url.us-east-1.on.aws/MouseLock)](https://github.com/Zeffuro/MouseLock)

Keeps FFXIV in right-click mouselook without requiring you to hold RMB. The cursor stays hidden and centered while locked, then automatically becomes usable again when you open bags, menus, chat, or other game windows.

## Build

```powershell
dotnet build .\MouseLock.csproj
```
